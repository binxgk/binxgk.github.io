Visual Studio Code

2019-07-06

vs code的特点：

  • 开源，免费
  • 自定义配置
  • 智能提示强大
  • 支持各种文件格式（html/jade/css/less/sass/xml）
  • 调试功能强大
  • 各种方便的快捷键
  • 强大的插件扩展
    以上是各位大佬的总结，也是软件的特色。

下载地址：

Visual Studio Code（VScode ）官网：
https://code.visualstudio.com/

Visual Studio Code（VScode ）github地址：
https://github.com/Microsoft/vscode

安装插件的方法：

  1. 按F1或Ctrl+Shift+p，输入extensions，点击第一个就可以
  2. ctrl + P 然后输入 >ext install
  3. 点击界面上的扩展图标

插件下载

插件官网：https://marketplace.visualstudio.com/
